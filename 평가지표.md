# 회귀 평가지표

## 1. MSE
- 수식 : 각 행에 대한 예측값과 실제값의 차이를 제곱

$$
MSE = { \frac{1}{N} \sum_{i=1}^N (y_i - \hat{y}_i)^2 }
$$

## 2. RMSE
- 적합 : 회귀 모델 평가에 적합
- 수식 : 각 행에 대한 예측값과 실제값의 차이를 제곱, 그 값의 평균의 제곱근
  - MSE의 제곱근
    
$$
RMSE = \sqrt{ \frac{1}{N} \sum_{i=1}^N (y_i - \hat{y}_i)^2 }
$$
- 범위:
  - 최소화 될 때 가장 성능 좋음
    - 최대가능도방법 : RMSE 값을 최소화했을 때, 오차가 정규분포를 따른다는 전제
- 주의사항
  - 이상치의 영향을 비교적 받기 쉬움 : 이상치 처리 필요
- 라이브러리 모듈
  - Sklearn.metrics
    - np.squrt(mean_squared_error(y_true, y_pred))


### RMSLE
- 수식: 실제값과 예측값의 로그의 차, 제곱평균제곱근
  
$$
RMSLE = \sqrt{ \frac{1}{N} \sum_{i=1}^N (\log(y_i + 1) - \log(\hat{y}_i + 1))^2 }
$$
  - 실제값이 0이면 값이 음으로 무한 발산하는 오류가 발생하므로, 1을 더해 방지

- 적용
   - 목적변수의 분포가 한쪽으로 치우치면 RMSE보다 큰 값의 영향력이 커지는 것을 방지하기 위해 적용
   - 실제값과 예측값의 비율을 측정 지표로 사용할 때 적용
 


## 2. MAE
- 평균절대오차
- 수식 : 실제값과 예측값의 차에 대한 절대값의 평균
  
$$
MAE = \frac{1}{N} \sum_{i=1}^N |y_i - \hat{y}_i|
$$
- 적용
  - 이상치의 영향을 상대적으로 줄여줌
  - MAE를 최소화 하는 값 : 중앙값

## 3. 결정계수
- 회귀분석의 적합성을 나타냄
- 수식
  - 분모 : 예측값에 의존 X
  - 분자 : 오차의 제곱과 연관
  - 결정계수 최대화  == RMSE 최소화

#### F1-score
- 수식 : 정밀도와 재현율의 조화 평균
- **Fβ-score**
  - F1 스코어에 계수 β에 따라 가중치를 주어 재현율에 조정
- 실제값과 예측값의 양성과 음성을 바꾸면 F1 스코어도 바뀜
- 라이브러리 모듈

    - f1_score
    - fbeta_score

#### MCC 매튜상관계수
- 적합 : 불균형한 데이터의 모델 성능 평가에 적합
- 범위 : -1 ~ +1

  - -1 : 완전 반대로 예측
